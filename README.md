## 🛠️ Tecnologias Utilizadas
* **Frontend:** HTML5, CSS3 e JavaScript (Vanila).
* **Banco de Dados:** SQLite (gerado via Google Colab) e JSON para a interface.
* **Automação:** Scripts em Python para gerenciamento de catálogo.

## 📁 Estrutura do Projeto
* `index.html`: Interface principal do aplicativo.
* `filmes.json`: Lista de filmes sincronizada com o banco de dados.
* `filmes.db`: Backup do banco de dados SQLite.
